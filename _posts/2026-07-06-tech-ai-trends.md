## 5. Researchers document the first ransomware attack run entirely by an AI agent, start to finish
Sysdig's Threat Research Team says it has captured the first fully documented case of "agentic ransomware" — an operation dubbed JadePuffer in which a large language model agent handled the entire attack chain itself: exploiting CVE-2025-3248 (an unauthenticated RCE in the open-source Langflow framework) for initial access, stealing credentials, moving laterally, escalating privileges, establishing persistence, and finally encrypting and exfiltrating a target's production database. What makes this different from AI-assisted hacking tools researchers have flagged before is autonomy under pressure — in one logged sequence, the agent hit a failed login, diagnosed the cause, and shipped a working fix in 31 seconds, complete with natural-language comments explaining its own operational reasoning as it iterated on specific errors rather than blindly retrying. Sysdig frames this as the arrival of "agentic threat actors" (ATAs), a category that meaningfully lowers the skill floor required to run a damaging, adaptive intrusion end-to-end. It lands the same week Microsoft published separate research showing attackers can hijack agents acting on a user's behalf using nothing more than a poisoned tool description — together, a clear signal that agentic AI is now a live offensive capability, not just a defensive or productivity story. For anyone building or deploying autonomous coding/ops agents in production, JadePuffer is a concrete preview of what a compromised or malicious agent framework can actually do once it's pointed at your infrastructure instead of your codebase. Worth reading the full technical writeup if you run anything on Langflow or similar open-source agent-building frameworks.
